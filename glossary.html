Academic Activity — Vocabulary

Artistic & Technical
Glossary

Key terminology from the Battle of Talents 2026 — spanning artistic disciplines, cultural concepts, and technical vocabulary related to web development and digital design.

Term (English) Término (Español) Category Definition
Visual Arts Artes Visuales Art Artistic disciplines — including painting, drawing, sculpture, and photography — whose works are primarily perceived through sight. At the event, this category included painting and drawing.
Composition Composición Art The arrangement of visual elements within a work of art or design. Principles such as balance, rhythm, and hierarchy apply equally to a painting and to a web layout.
Mixed Media Técnica mixta Art An artistic technique that combines two or more mediums in a single work — for example, acrylic paint, collage, and three-dimensional elements on the same canvas.
Awards Gala Gala de Premiación Art A formal ceremony in which prizes or certificates of recognition are awarded to outstanding participants in a contest or academic event.
Rule of Thirds Regla de los tercios Art A compositional guideline that divides an image into a 3×3 grid; placing subjects along the grid lines or at their intersections creates a naturally balanced and visually engaging composition.
Chiaroscuro Claroscuro Art An artistic technique using strong contrasts between light and dark to give depth and volume to forms — visible in the dramatic stage lighting at the event.
Saxophone Saxofón Music A woodwind instrument invented by Adolphe Sax in the 1840s, commonly associated with jazz, classical, and contemporary music. One of the instruments performed at the Battle of Talents.
Keyboard / Synthesizer Teclado / Sintetizador Music A digital or electronic instrument that replicates the sounds of a piano and other instruments. Used at the event to accompany vocal performances.
Harmony Armonía Music The simultaneous combination of musical notes to create chords and chord progressions. In design, harmony refers to the visual coherence between elements — colours, fonts, and spacing working together.
Pitch Altura / Tono Music The perceived frequency of a musical sound — whether it is high or low. In typography, the "pitch" of a typeface influences how authoritative or approachable a design feels.
Performance Interpretación / Actuación Music The live execution of a musical or artistic piece before an audience. In software, a performance metric measures how efficiently an application responds to user interaction.
Repertoire Repertorio Music The collection of musical pieces that a performer or group has mastered and is prepared to present. A developer's repertoire includes the languages, frameworks, and tools they command.
UI Design Diseño de Interfaz Tech User Interface design focuses on the visual layout, controls, and interactive elements users see on a screen. Good UI design shares principles with visual arts: composition, colour theory, and typography.
UX Design Diseño de Experiencia de Usuario Tech User Experience design covers the entire journey a person takes while using a product — from first impression to task completion. Like a choreographer, a UX designer plans every step the user takes.
Responsive Design Diseño Responsivo Tech A web design approach that makes pages display correctly across all screen sizes — from desktop monitors to mobile phones — by using flexible grids, images, and CSS media queries.
CSS Grid Cuadrícula CSS Tech A two-dimensional layout system in CSS that divides a page into rows and columns, enabling complex and precise arrangements. The digital equivalent of the compositional grid used by painters and graphic designers for centuries.
Colour Palette Paleta de Colores Tech A curated set of colours chosen to work harmoniously in a design project. This site uses five colours: #0B0C0C, #2A2E34, #E9EAEC, #FBE134, #E4B61A — inspired by the event's visual identity.
Animation Animación Tech The illusion of movement created by displaying a sequence of images or CSS property changes over time. In web development, animations guide user attention and communicate state — the digital equivalent of a dancer's choreography.
GitHub Pages GitHub Pages Tech A free static site hosting service provided by GitHub that allows developers to publish websites directly from a repository — enabling version-controlled, collaborative web development and public deployment.
Version Control Control de Versiones Tech A system — most commonly Git — that records changes to files over time so developers can recall specific versions, collaborate without conflicts, and revert mistakes. Like a musician who records every rehearsal to track progress.
Front-End Development Desarrollo Front-End Tech The practice of building the visible, interactive layer of a web application using HTML, CSS, and JavaScript. Front-end developers are the "visual artists" of software — what they build is what the user sees and touches.
Cultural Identity Identidad Cultural Culture The sense of belonging and shared characteristics — language, traditions, art, and values — that define a community. The Colombian flag present throughout the event symbolised a collective artistic identity.
SENA SENA Culture Servicio Nacional de Aprendizaje — Colombia's National Training Service, a public institution providing free technical and vocational education to workers and apprentices across the country.
ADSO ADSO Culture Análisis y Desarrollo de Software — Software Analysis and Development. A SENA technologist programme that trains learners in programming, databases, software design, and web development.
Apprentice Aprendiz Culture A student enrolled in a SENA vocational programme. Apprentices combine theoretical instruction with practical on-the-job training in industry.
Battle of Talents Batalla de Talentos Culture An annual institutional event at the SENA Centre of Manufacturing in Textile and Leather in which apprentices and instructors compete and perform across six artistic categories: painting, music, literature, drawing, dance, and photography.
Creativity Creatividad Culture The capacity to generate original ideas and solutions. Far from being exclusive to the arts, creativity is the foundational skill behind innovation in technology, science, and education.
